# Planet Hunter

Using AI to Find the Stars That Could Be Home.

## About the App

Planet Hunter is an immersive, science-fiction interactive web application that integrates Machine Learning 
with user experience design to classify celestial bodies. Acting as the core intelligence framework for a 
deep-space scouting vessel, the system dynamically analyzes real-world stellar properties like size and heat. 
The core objective is to isolate habitable, Sun-like stars capable of serving as humanity's next cosmic sanctuary 
(Nova Terra). It provides an educational yet engaging experience for users to explore stellar taxonomy through 
a data-driven narrative approach.

## Features

* Dynamic On-the-Fly Training: Machine learning models are trained instantly from raw storage directly into volatile system memory (RAM) at server runtime, preventing file-version mismatches.
* Astrophysical Boundary Modeling: Classification criteria align with the mechanics of the Hertzsprung-Russell (H-R) Diagram to group stars based on heat indices and absolute magnitude.
* Narrative Driven User Interface: Provides contextual story telemetry feedback driven by the built-in spaceship assistant systems, Marry and Rocky.
* Interactive Attribute Manipulation: Users can slide values for star dimensions to witness real-time predictive shifts from the classifier engine.
* Responsive Design Layout: Clean, modern style dashboard tailored perfectly for different screen viewing configurations.

## How to Run the Project

1. Clone or download this project workspace repository down to your local machine directory.
2. Ensure you have Python installed on your system.
3. Open your system terminal or command prompt, navigate to the project root folder, and install the required modules by executing the command: pip install flask pandas scikit-learn numpy
4. Make sure your local knowledge base dataset file named star_datatype.csv is placed inside the same folder directory.
5. Boot up the local web development server by running the command: python app.py
6. Open your web browser and navigate to the local hosting address generated by the framework: http://127.0.0.1:5000
